# FounderOS Roadmap

## Current Foundation

FounderOS currently has:

- Supabase authentication
- Supabase database persistence
- OpenAI project chat
- AI structured object extraction
- Projects
- Tasks
- Notes
- Decisions
- Risks
- Roadmap items
- Idea Vault
- AI Idea Analysis
- Turn Idea into Project
- Project Review Mode
- Weekly Review Mode
- Knowledge Graph / Linked Memory
- Global Command Bar
- File Upload + Project Memory

## Core Loop

Capture → Organise → Plan → Execute → Review → Improve
