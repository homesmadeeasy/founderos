# # FounderOS Test Prompts

Purpose:

This file is used to test whether FounderOS gives useful, practical responses for realistic users.

FounderOS should help users:

- capture ideas

- turn ideas into projects

- create tasks, notes, decisions, risks and roadmap items

- review progress

- search memory

- avoid generic AI answers

---

## Test 1: Vague Founder Idea

Where to test:

Idea Vault

Prompt:

I want to build an app but I’m not sure what it should be. I like AI, productivity and helping people finish projects.

Expected behaviour:

- Helps narrow the idea

- Challenges vagueness

- Suggests possible target users

- Suggests a realistic MVP

- Identifies risks

- Gives validation steps

Score:

Understanding: /5

Usefulness: /5

Specificity: /5

Would a real user know what to do next? /5
